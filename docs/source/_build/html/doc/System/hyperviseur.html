Hyperviseur - OnSphere¶

Introduction¶

OnSphere est un framework puissant qui sert de solution pour l’automatisation de diverses tâches et processus. Son objectif premier est de connecter de manière transparente plusieurs systèmes qui n’ont pas forcément été conçus à l’origine pour fonctionner ensemble. Cette flexibilité fait d’OnSphere un choix idéal pour un ensemble varié de problèmes d’automatisation.

Dashboard utilisateur¶

OnSphere permet aux intégrateurs de créer des tableaux de bord pour les utilisateurs et les groupes. Un tableau de bord dépend de ce que l’intégrateur souhaite afficher à ses utilisateurs. Exemple de ce qui peut être affiché :

  • Afficher la liste des alarmes reçues par le système

  • Permettre à l’utilisateur d’interagir avec la sécurité de l’accès physique (armer, désarmer) et de gérer les horaires.

  • Afficher les valeurs historiques à l’aide de filtres et de requêtes avancées

  • Gérer les services de l’utilisateur et les tâches programmées à effectuer

Cas d’utilisation¶

OnSphere sera dans notre cas utilisé afin de gérer la communication entre tout les appareils domotique du bâtiment.
