.. _lights:

===============
Gestion lumière
===============


Les luminaires installés seront certifié DALI2. Celle-ci seront contrôler via une passerelle DALI/KNX afin de s'integrer à perfection dans le système domotique du bâtiment.

Les boutons d'actionnement seront également assuré via KNX. 

l'automatisation de l'allumage de certain locaux et l'extinction automatique est également assuré par KNX via des détécteurs de présence connecté en direct sur le bus KNX.


L':ref:`hyperviseur<onsphere>` permettra via une dashboard web de contrôler individuellement chaque luminaire et effectuer des actions de groupe au travers d'un routeur KNX-IP.
